# Agent Instructions

このリポジトリは、中小企業や1人事業者がAI-assisted workflowsを設計、構築、運用するためのOSSです。

## Core Rules

- マーケットイン思考で、現場課題から逆算する
- 抽象論だけで終わらせず、実務テンプレートに落とす
- 不安をあおらない
- 薄利多売を前提にしない
- 労働集約を前提にしない
- AIが担う作業と、人が担う判断を分ける
- 法的、倫理的、個人情報に関わる判断は人間レビューを前提にする

## Output Rules

- Markdown中心で、差分管理しやすく書く
- 1ファイルに詰め込みすぎず、役割ごとに分割する
- READMEはOSSとして初見の人が理解できる内容にする
- テンプレートは入力欄、判断基準、出力例を含める
- 例は中小企業、1人社労士、Web支援、AI活用支援の文脈に寄せる
