# Contributing

このプロジェクトは、中小企業や1人事業者が実務で使えるAI業務設計テンプレートを育てるためのOSSです。

## Contribution Areas

- 業務分解テンプレートの改善
- 業種別ユースケースの追加
- AIと人の責任境界チェックリストの改善
- セキュリティ、個人情報、レビュー観点の追加
- MarkdownテンプレートからHTMLや提案書へ展開する例の追加
- Claude Code / Codex で使いやすい運用ルールの改善

## Contribution Principles

- 不安をあおらない
- 薄利多売を前提にしない
- 労働集約に戻さない
- 現場で使える粒度まで具体化する
- AI導入ありきではなく、顧客課題から逆算する
- 人が責任を持つべき判断をAIに丸投げしない

## Pull Request Checklist

- 目的と対象ユーザーが明確である
- 実務で使う入力項目と出力項目がある
- 人とAIの役割分担が曖昧になっていない
- 専門用語が多すぎない
- 中小企業や1人事業者でも使いやすい
